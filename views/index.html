Image Search API

Two main functions to this API:

  1. Takes a search string and returns a list of images with metadata
  2. Shows the latest search

  1. Image Search

    Template of usage: https://closed-grasshopper.glitch.me/api/imagesearch/[search]?offset=[pagination]

    Example of usage: https://closed-grasshopper.glitch.me/api/imagesearch/lolcats%20funny?offset=10

  2. Latest Search

    3. Template of usage: https://closed-grasshopper.glitch.me/api/latest/imagesearch/
